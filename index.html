Техническая спецификация программного обеспечения для ConveNote

Обзор: Наше приложение для Android будет представлять собой блокнот с функциями для быстрого и удобного ведения заметок, а также несколькими дополнительными функциями. Он будет включать в себя функции chatGPT , калькулятор и другие инструменты для упрощения написания текстов.

Функциональность: 1. Блокнот: - Пользователи могут создавать, редактировать и сохранять заметки в приложении. - Пользователи могут легко искать заметки. - Заметки будут организованы так, чтобы самые последние из них отображались вверху и упорядочивались по дате. 2. ChatGPT: - Функция, которая позволяет пользователям вести беседу с помощью chatGPT. - Пользователи могут ввести вопрос или тему, и chatGPT предоставит текстовый ответ. - ChatGPT будет использовать обработку естественного языка. 3. Калькулятор: - Пользователи могут использовать встроенные вычисления и формулы. - Калькулятор будет встроен в приложение. Дизайн: 1. Начальный экран: - Простой, удобный интерфейс с возможностью открытия блокнота. - У пользователей будет возможность добавить новую заметку или просмотреть предыдущие заметки. 2. Блокнот: - Экран для создания или редактирования заметок. - Возможность сохранять заметки на устройстве или загружать в облачное хранилище. 3. ChatGPT: - Алгоритм, который интегрирует ответ chatGpt. 4. Калькулятор: - Простые математические параметры, которые пользователь может использовать в приложении. Техническая информация: 1. Языки программирования: Java и XML. 2. IDE: Android Studio.
